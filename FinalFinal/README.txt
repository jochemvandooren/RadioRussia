Alle algoritmes produceren een ingekleurde SVG map. Zie de print statements voor meer info na het runnen van de files.

In de map Greedy staan 2 algoritmes. 
Greedy.py en GreedyLeastOptions.py.
Greedy.py is ons basic greedy algoritme en GreedyLeastOptions.py is ons greedy algoritme met een heuristiek.

In de map Eventowers staat 1 algoritme. 
Eventowers.py
Eventowers.py is ons depth first algoritme met heurstiek. (least options first)

Bovenaan de python files staan instructies voor het algoritme, bekijk altijd de output!
